# armstrong_number
Python program to check Armstrong number using basic loop logic.

# Armstrong Number Checker

An Armstrong number is a number that is equal to the sum of its digits 
raised to the power of the number of digits.

Example:
153 → 1³ + 5³ + 3³ = 153 (Armstrong Number)

## 🚀 How It Works
1. Count the number of digits.
2. Extract each digit using modulus (%).
3. Raise each digit to the power of total digits.
4. Compare the result with the original number.

## 🖥️ Input
Single integer from user input.

## 📤 Output
Prints:
- "Armstrong" if true
- "Not Armstrong" if false

## 📌 Example

Input:
153

Output:
Armstrong

Input:
123

Output:
Not Armstrong
